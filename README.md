



🍔 Hamburgueria Dev - Sistema de Pedidos Online
Uma interface moderna, rápida e responsiva para facilitar o atendimento da sua hamburgueria.



📌 Sobre o Projeto
Este é um projeto de um site institucional para uma hamburgueria, focado na experiência do usuário e na facilidade de realizar pedidos via WhatsApp. O site conta com um menu dinâmico, verificação de horário de funcionamento e um carrinho de compras funcional.



🚀 Funcionalidades
Menu Digital: Visualização clara dos itens com preços e descrições.

Carrinho em Tempo Real: Adicione e remova itens com facilidade.

Integração com WhatsApp: O pedido final é enviado formatado diretamente para o atendente.

Status de Funcionamento: Sistema inteligente que verifica se o estabelecimento está aberto no momento.

Responsividade: Totalmente adaptado para celulares, tablets e computadores.



🛠️ Tecnologias Utilizadas
O projeto foi construído utilizando o "trio de ferro" do desenvolvimento web:

HTML5: Estruturação semântica do conteúdo.

CSS3: Estilização moderna 

JavaScript: Lógica do carrinho, cálculos de preços e integração com API.



📂 Como visualizar o projeto
Para explorar o código ou rodar o projeto localmente, siga a estrutura das pastas:

index.html: Contém toda a estrutura e elementos da interface.

style.css: Responsável pelo design, cores e layout.

script.js: Onde a mágica acontece (lógica da interações com o usuário).
